# 화요일엔 나도 SQL 요리사!

## 소개

## 스터디 방식
<img src = "https://user-images.githubusercontent.com/101805118/162907359-d6aaa367-aeb8-4617-9fb4-3f20561b297b.png" width = 200/>

- **<데이터 분석을 위한 SQL 레시피>** 도서를 응용하여 발표합니다.<br>
